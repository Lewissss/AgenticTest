All API calls are mocked but we include reference material for selectors and tests.

| Endpoint | Description |
| --- | --- |
| `GET /runs` | List available runs |
| `POST /traces/compile` | Compile a trace into a deterministic Bun test |
| `POST /playbooks/execute` | Kick off a playbook |
